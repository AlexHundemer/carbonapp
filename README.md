# Project Name

A modern, AI-assisted web application built using a **vibe coding** workflow.

This project is designed to be:
- safe for non-programmers
- guided by clear constraints
- powered by real cloud infrastructure
- built incrementally with AI assistance

The goal is to move from idea → working product without unnecessary complexity.

---

## What This Project Is

- A React-based web application
- Built with Vite for fast local development
- Uses Firebase (Firestore) for real cloud data persistence
- Developed with AI agents (Anti-gravity + Stitch) under strict guardrails

---

## What This Project Is NOT

- ❌ Backend-heavy system
- ❌ Microservices architecture
- ❌ Framework experimentation playground
- ❌ Undocumented magic
