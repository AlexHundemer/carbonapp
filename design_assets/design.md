# Design

This document is the **visual and UX source of truth**.

If design is unclear, refer here before making assumptions.

---

## Design Goals

- Simple
- Calm
- Minimal
- Intuitive
- Low cognitive load

---

## Visual Style

- Neutral color palette
- Generous spacing
- Clear typography
- Minimal animation

---

## UI Principles

- Fewer buttons > more buttons
- Obvious actions > clever actions
- Text clarity over visual flair

## Design Changes

Design changes must:
- Improve clarity
- Reduce friction
- Be intentional
